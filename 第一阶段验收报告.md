## 4. MiniMax 中文 word_streaming 实测样例

```
request: model=speech-2.8-hd
         text="在川西高原的风里,有一座叫炉霍的地方。"
         language_boost=Chinese
         voice_id=male-qn-jingying
         subtitle_enable=true
         subtitle_type=word_streaming
         stream=true

response SSE chunks: 12
  status=1 (intermediate): 11  ← 累计+重叠, audio 跳过
  status=2 (final):        1  ← 唯一终态, audio 用这个
audio_length: 10425 ms (10.43s)

sample word entries:
  {word: "在", begin: 42ms,  end: 128ms,  charBegin: 0, charEnd: 1}
  {word: "川", begin: 128ms, end: 298ms,  charBegin: 1, charEnd: 2}
  {word: "西", begin: 298ms, end: 469ms,  charBegin: 2, charEnd: 3}
  {word: "高", begin: 469ms, end: 597ms,  charBegin: 3, charEnd: 4}
  ...

关键发现:
  ✓ 中文 word 是字级 (word_end - word_begin = 1)
    不同于英文的"音节碎片", 不需要合并逻辑
  ✓ 字段命名与英文 100% 一致:
    word / time_begin / time_end / word_begin / word_end
  ✓ 每个 chunk 都带该句累计 words, 按 Math.round(time_begin) 去重
  ✓ base_resp.status_code = 0 = success
```

## 5. 实际跑过的命令 (可复现)

```bash
# 1. 探测 (新 vendor/新语言时建议先跑)
node scripts/probe-minimax-tts.mjs

# 2. 跑音频 + 写回 shots.json
node scripts/generate-audio.mjs
#   → 4 个 shot 各自 start/end, 总时长 11.93s
#   → 写 public/audio/luhuo-heritage-001.mp3 (169 KB)
#   → 写 public/audio/luhuo-heritage-001.words.json (55 个字)
#   → 备份 content/shots.json.bak

# 3. 渲染
node scripts/build-video.mjs --skip-audio
#   → npx remotion render src/index.ts LuhuoMain out/luhuo-main.mp4
#   → 358 帧 / 11.93s / 728 KB

# 4. 一条命令搞定 (audio + render)
npm run build
```

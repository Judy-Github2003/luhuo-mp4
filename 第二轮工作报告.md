## 2. shots.json 里 image / video 示例写法

**image shot**:
```json
{
  "id": "shot-02",
  "kind": "scene",
  "duration": 5,
  "zhNarration": "这里的天空很近,云走得很慢。",
  "zhSubtitle": "这里的天空很近,云走得很慢。",
  "boSubtitle": "",
  "visual": {
    "type": "image",
    "src": "images/shot-02.png",
    "showTitle": true
  }
}
```

**video shot**:
```json
{
  "id": "shot-03",
  "kind": "scene",
  "duration": 4,
  "zhNarration": "山脊上飘着五彩的经幡。",
  "zhSubtitle": "山脊上飘着五彩的经幡。",
  "boSubtitle": "",
  "visual": {
    "type": "video",
    "src": "videos/shot-03.mp4",
    "muted": true,
    "showTitle": true
  }
}
```

**约定**:
- `visual.type` 切换类型 (`color-block` / `image` / `video`)
- `visual.src` 是**相对 `public/`** 的路径
- `visual.muted` (video 模式) — 默认 `true`，因为旁白是独立 audio track
- `visual.showTitle` (image / video 模式) — 默认 `false`，设 `true` 时叠一层半透明标题卡
- `visual.background` / `visual.accent` (color-block 模式) — 颜色和装饰色

---

## 3. ShotVideo.tsx 三种类型支持

| `visual.type` | 实现 | 组件 |
|---|---|---|
| `color-block` | 纯色 + 斜线 + 标题 | 内部 `ColorBlock` |
| `image` | 满铺 cover | Remotion `<Img src={staticFile(src)} />` |
| `video` | 满铺 cover + 默认静音 | Remotion `<OffthreadVideo src={staticFile(src)} muted />` |
| 其它 / 缺失 | 兜底 color-block | `ColorBlock` |

**回退策略 (关键设计)**: 不在 React 渲染时做素材探测（Remotion 是静态渲染，`useEffect` 不可靠），而是在 **build 时**由 `verify-assets.mjs` 扫描 `visual.src` 写到 `visual.effectiveType`。渲染时 `ShotVideo` 优先读 `effectiveType`（已校验过），缺失/未知类型 → 写 `'color-block'`。

**回退触发条件** (任一):
- `visual.type` 是 `color-block` → `effectiveType: 'color-block'`
- `visual.type` 是 `image` / `video` 但 `src` 缺失 → 报警 + `effectiveType: 'color-block'`
- `visual.type` 是 `image` / `video` 但 `src` 文件不存在 → 报警 + `effectiveType: 'color-block'`
- `visual.type` 是未知值 → 报警 + `effectiveType: 'color-block'`

报警写到 `verify-assets` 控制台输出，**不阻断 build**。

## 6. 新增 / 改动的代码片段（关键逻辑）

### 6.1 ShotVideo.tsx 关键分支

```tsx
const effectiveType = visual.effectiveType ?? visual.type ?? 'color-block';

if (effectiveType === 'image' && visual.src) {
  return (
    <AbsoluteFill style={{background: '#000', overflow: 'hidden'}}>
      <Img src={staticFile(visual.src)} style={{width: '100%', height: '100%', objectFit: 'cover'}} />
      {(visual.showTitle ?? false) && <TitleOverlay ... />}
    </AbsoluteFill>
  );
}

if (effectiveType === 'video' && visual.src) {
  return (
    <AbsoluteFill style={{background: '#000', overflow: 'hidden'}}>
      <OffthreadVideo
        muted={visual.muted ?? true}
        src={staticFile(visual.src)}
        style={{width: '100%', height: '100%', objectFit: 'cover'}}
      />
      {(visual.showTitle ?? false) && <TitleOverlay ... />}
    </AbsoluteFill>
  );
}

// 兜底 color-block
return <ColorBlock bg={...} accent={...} title={title} subtitle={subtitle} />;
```

### 6.2 verify-assets.mjs 关键逻辑

```js
for (const shot of content.shots) {
  const visual = shot.visual ?? {};
  const requested = visual.type ?? 'color-block';
  if (requested === 'color-block') { visual.effectiveType = 'color-block'; continue; }
  if (requested !== 'image' && requested !== 'video') { visual.effectiveType = 'color-block'; continue; }
  if (!visual.src) { visual.effectiveType = 'color-block'; continue; }
  try {
    await access(resolve(publicDir, visual.src));
    visual.effectiveType = requested;  // image / video
  } catch {
    visual.effectiveType = 'color-block';  // 兜底
  }
}
```
